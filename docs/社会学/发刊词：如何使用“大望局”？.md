 1. 祝《大望局》能成为你的“电子闺蜜”。

 2. 男人最好的医美是眼泪。

 3. 也许吧，不是勇敢的人先享受世界，而是“觉少”的人先享受啊。

 4. 北野武在电影《红鳉鱼》中的台词：“现实就是答案，就算抱怨生不逢时，社会不公，也不会有任何改变。现实就是现实，要理解现状并且分析，在那其中一定会有导致现状的原因，对原因有了充分认识之后再据此付诸行动就好。连现状都不懂得判断的人，在我看来就是白痴。”——北野武本人也经常表达类似的人生观点。
